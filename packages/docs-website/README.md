# Spacetime Fabric Documentation Website

Interactive documentation and demo site for the `react-spacetime-fabric` package.

## Development

```bash
# Install dependencies (from root)
pnpm install

# Run dev server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Features

- Interactive physics simulation with live controls
- Comprehensive documentation of all configuration options
- Built-in physics explanations
- Mobile-responsive design

## Deployment

This site can be deployed to Vercel, Netlify, or any static hosting service.

```bash
# Build the site
pnpm build

# Output will be in dist/
```
